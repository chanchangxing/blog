风语飘飘面试总结

  1. 了解前几家公司详情：感觉三年经验三家公司这个履历是比较糟糕，但是我的离职的理由还是比较硬码的。所以应该来说没有什么问题。
  2. 他问的问题就是这个问题“Android各个版本最重要的更新”。感觉就是这道题目没有打好，其他问题都是很简单的面试题，所以这道题目要再去找一下。
  3. 就不仔细聊关于他对RN的错误理解了，在聊webview的时候我对于性能和安全性时候对这里的知识点有点遗忘了。
  4. 这里有个需要思考的东西，如果我内心已经拒绝的公司，我是表明自己的意向呢，还是要吊着他呢。

总的来说这是一次非常糟糕的面试经历，所以也只能写那么多，想着如果有好的公司的面试经历，总结的应该会更多吧。
