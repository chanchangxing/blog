Android各版本更新

5.0

  1. 加入了Material Design
  2. ART 运行时取代 Dalvik 成为平台默认设置。
    • 预先 (AOT) 编译
    • 改进的垃圾回收 (GC)
    • 改进的调试支持
  3. 加入了新的 Job Scheduling
  4. webview会阻止混合内容和第三方Cookie。系统现在可以智能地选择要绘制的 HTML 文档部分。

6.0

  1. 运行时权限检查（Runtime Permisssions）
  2. 低电耗模式和应用待机模式
  3. 移除Appache的HTTP Client，请使用HttpURLConnection类进行替换

7.0

  1. 多窗口支持
  2. 添加了即时 (JIT) 编译器
  3. 目录被限制访问
  4. 应用间共享文件
  5. APK signature scheme v2，它能提供更快的应用安装时间和更多针对未授权 APK 文件更改的保护
  6. 快捷方式

8.0

  1. 画中画
  2. webview API

9.0

  1. 刘海屏适配
