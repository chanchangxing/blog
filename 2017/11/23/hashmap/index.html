hashmap

1、hashmap的put方法

put方法会先通过a.hashCode()方法来计算a的hashcode值来找到他的散列码，然后在bucket中寻找是否已经有这个散列码的键值对的List(bucket在编程思想中被称为‘桶位’，他是一个数组)。

​ 如果键值对List有的话，然后去遍历这个List，并且将里面的键和a进行equal比较。如果equal返回为true，则将该值改为b，否则将该键值对加入这个List。

​ 如果没有这个List的话，说明还没有散到这个点，那就重新创建一个List，并将键值对放到这个List里面。

2、hashmap的get方法

get方法和put方法差不多，都是会去找hashcode的散列码和调用equal方法。如果都没有找到的话，会返回null
